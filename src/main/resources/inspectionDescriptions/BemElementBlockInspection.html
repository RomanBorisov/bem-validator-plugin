Validates that BEM element classes are used together with their parent block classes.

Ensures that element classes (e.g., block__element) are always used with their parent block class in the same HTML element.

Rules

  • Element class must be used with its parent block class
  • Both classes must be in the same HTML element
  • Block names must match

Examples



✓ <div class="menu menu__item">
✓ <div class="form form__input form__label">
✓ <div class="header header__logo header__logo--small">


✗ <div class="menu__item">                
✗ <div class="header menu__item">         
✗ <div class="nav"><span class="menu__item">

Note: This inspection helps maintain proper BEM structure and ensures that elements are always semantically connected to their blocks through class names.
