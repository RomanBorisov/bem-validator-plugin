Validates that BEM modifier classes are used together with their base block or element classes.

Ensures that modifier classes (--modifier) are always used with their base classes (block or block__element) in the same HTML element.

Rules

  • Block modifier needs its block class
  • Element modifier needs its element class
  • Both classes must be in the same element

Examples



✓ <div class="button button--primary">
    Primary button
</div>


✓ <div class="menu menu__item menu__item--active">
    Active menu item
</div>


✓ <div class="form__input form__input--large form__input--disabled">
    Large disabled input
</div>


✗ <div class="button--primary">         
✗ <div class="menu__item--active">     
✗ <div class="button input--large">

Note: This inspection helps maintain the proper relationship between modifiers and their base classes, ensuring that your BEM classes correctly represent the component's state or variation.
