Validates that class names follow BEM (Block Element Modifier) naming conventions.

Ensures CSS class names follow BEM methodology naming conventions for better maintainability and reusability.

Rules

  • Characters: lowercase letters, numbers, hyphens, underscores
  • Element: two underscores (block__element)
  • Modifier: two hyphens (block--modifier)

Examples



✓ header
✓ menu__item
✓ button--primary
✓ form__input--disabled


✗ Header              
✗ menu___item         
✗ form__input__label  
✗ block--mod1--mod2
