Validates that BEM elements are not nested too deeply.

Ensures that BEM elements don't use multiple levels of nesting in their class names, which can lead to maintainability issues.

Rules

  • Only one level of element nesting is allowed
  • No chaining of elements with double underscores

Examples



✓ <div class="block">
✓ <div class="block__element">
✓ <div class="block__element--modifier">


✗ <div class="block__element__subelement">
✗ <div class="block__element__subelement--modifier">
✗ <div class="block__element__subelement__deepnested">
